# Hello World with Gemini Ai

### Created environment
```bash
python -m venv hello_world
```

### Active env (windows)
```bash
hello_world\Scripts\activate
```

### Active env (Linux)
```bash
source hello_world/bin/activate
```

### Install dependencies
```bash
pip install google-generativeai
```

### Execute project
```bash
python hello_world.py
```
